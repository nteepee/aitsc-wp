## Image Specifications for Designers

### File Formats
```
Hero images: JPG (quality 85%) OR PNG (if transparency)
Product photos: PNG (transparent background) OR JPG (white background)
Diagrams: PNG (lossless compression)
Screenshots: PNG (crisp text)
```

### Dimensions
```
Hero (desktop): 1200x800px minimum, 1920x1080px preferred
Hero (mobile): 800x1200px (vertical, 4:5 aspect ratio)
Feature images: 600x400px minimum
Testimonials: 300x300px (square, circular crop)
Spec diagrams: 1000x700px minimum
Gallery: 800x600px minimum
```

### File Sizes
```
Target: Under 200KB per image (for web performance)
Hero: Up to 300KB acceptable
Mobile: Under 100KB per image
Use: ImageOptim, TinyPNG, or Squoosh for compression
```

### Alt Text Templates
```
Product: [Product name] [feature] [context]
Example: "7-inch LCD seat belt monitoring display with color-coded seat status"

Stock photo: [Subject] [action] [location] [purpose]
Example: "School bus driver checking rearview mirror before departure, interior view"

Diagram: [Subject] showing [details] [purpose]
Example: "Coach bus seating layout diagram with 4-row configuration and seat numbers"
```
